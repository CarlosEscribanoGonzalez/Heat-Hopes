# Heat & Hopes

**Heat & Hopes** is a space-themed platformer game developed in Unity during my third year of university for the course **Human-Computer Interaction and Usability**.

### About the Project

The objective of this project was to create a game with a minimum level of playability that adhered to **adaptability** and **usability standards**. The game was designed to be clear, intuitive, and easy to use, while also providing settings that allow users to adjust the gameplay to their preferences, improving the overall experience.

### Features
- Space-themed platformer gameplay.
- Focus on adaptability and usability standards.
- User-adjustable settings to enhance gameplay and experience.

### Technologies
- **Unity**
- **C#**
